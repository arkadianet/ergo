Heads up: this is the full Scala-node OpenAPI spec. ergo-rust-node implements a large subset — /info, /blocks/*, /transactions/* (including the /unconfirmed pool queries and /bytes submit), /peers/*, /utils/seed, /utxo/genesis and /utxo/byId/{id}, the integrated /wallet/* (always mounted; browser UI at /wallet/ui), plus — when the matching subsystem is enabled — /blockchain/* and /mining/*. Everything else returns 404 (notably the rest of /utils/*). See /swagger/native for the Rust-native /api/v1/* API, or / for the live dashboard.
