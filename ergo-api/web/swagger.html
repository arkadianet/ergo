Heads up: this is the full Scala-node OpenAPI spec. Almost all of it is mounted; the gaps that still return 404 are POST /script/executeWithContext, POST /mining/candidateWithTxs, POST /utxo/getBoxesBinaryProof, and the Scala-shape /transactions/unconfirmed/{inputs,outputs}/… box lookups (this build serves the related POST /transactions/unconfirmed/by* tx filters instead). Config-gated families (/blockchain/*, /mining/*) are unmounted when their subsystem is off — see the spec description below. See /swagger/native for the Rust-native /api/v1/* API, or / for the live dashboard.
